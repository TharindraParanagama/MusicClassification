What is ClassifyMe?

This is a web based tool created for musicians to determine the rating of their song(s) in a convenient manner.In the development of the tool we have used meaningful high level features that will help determine the rating at a reasonable accuracy.Since this tool is a free application.go ahead and play with it.

How to use the system?

A step by step guide to obtain the rating for a sample song is as follows: The system is compatible only with audio files with .mp3 oor .wav extension only.Therefore make sure the audio file you are inputting to the system is in a compatible format.Next the song should be named in according to this convention "songName@Anything.mp3/.wav".It's of vital importance that the @ is avaiable in the name as the splitting is based on the @ symbol to derive the song name.further giving a distinct and meaningful name for your song will also help you when assessing the prediction results.After the song(s) is properly labeled the directory where the song /song are held can given to the text box above the extract button to extract the relevant features.As the prediction is based on these features the feature extraction should be performed first prior to any other operation.when the features have been extracted it will be saved to a csv file with the relevant names of the songs.If your feature ectraction was successful a pop up will alert the sucess .The path of the file which holds the extracted features must then be given to the textbox above the prediction button.This will prompt an popup which shows the rating of each song by name.

Helpful information for assesment:

when using the system the user can first determine a song which he/she likes and give it to the system to determine the rating given by the system.next the we have provided the user with the ranges in which each feature varies for each rating .therefore the user can give there song to the system and determine the label.then based on the label if there's a mismatch the user can check in which features he/she is out of range and determine necessary precautions for improve ment.

possible suggestions in improving the current state if mismatch exist.


adding different instruments to the song. adding more people to make it a group singing rather than solo. changing the environment in which the song is recorded.(indoor or outdoor)
